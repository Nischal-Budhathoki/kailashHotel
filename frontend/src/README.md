# Hotel Kailash Parbat
 A react + typescript hotel management system web-application to browse, guest, manage rooms and hr payroll, all in one. 

# Tech Stack
  - React
  - Typescript
  - html
  - tailwind css
  - vite
  - axios
  - react-router dom

# Installation
  - npm create vite@latest
  - git
  - docker 
  - npm install
  - react-icons

# Running the application
  - npm run dev
  - docker run 
  - git clone / git init

# Build for production
  - npm run build 

# features
 - guest can inquire
 - room booking system
 - room availability checking
 - staff and hr payroll management
 - admin dashboard
 - responsive ui for all devices

# Project Structure
src/
├── components/
├── pages/
├── services/
├── hooks/
├── routes/
└── App.tsx
